# Game-Theory-
Game Theory Project for the Iterated Prisoner's dilemma on Matlab.

This project is about materialization an Imitation and Fitness Dynamics evolutionary tournament. It attemps to replicate the results produced by the paper shown in the Report reference No. 4. It explores a variety of Game Theory strategies for the Iterated Prisoner's dilemma, following the structure of an Axelrod Tournament. For more details inspect the PDF.
